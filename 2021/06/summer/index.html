暑假要做的事

 2021.6.28  2021.6.30  雜文  1000  2 分鐘

停課也有好一陣子了，第二、第三次段考也都取消了，而星期五的結業式完後，準備迎接的，便是長達兩個月的暑假。暑假這段時間也蠻長的，暑假作業應該也一下就寫完了(應該吧？)，剩下的時間便可以拿來做其他的事。為了避免我這段時間過得太廢，所以我決定暑假的時候要開始做一些平常不會去做的事。所以下面是我在暑假想要去做或嘗試的事。

習慣

每天花點時間看書

不知從幾時開始，我很難去沈浸在書中的世界，所以我這個暑假，我每天都要花點時間看書，不管什麼書都好。

可以的話，我會希望我能把這些書看完:

  • 嫌疑犯X的獻身
  • Headspace 冥想正念手冊
  • 毛姆小說選集
  • 如何閱讀一本書

每天運動三十分鐘

在家裡悶了那麽久，也沒怎麼運動，所以暑假要多運動才行。

目前的話，可能只能做仰臥起坐、伏立挺身、深蹲、慢跑而已，不知道什麼時候才能打球。

九點半睡覺，七點起床

在停課以前，我平常也就十點多、大不了十點半就去睡了，但停課以來幾乎都拖到十點半甚至十一點，所以我要更早去睡覺。

寫子彈筆記

我其實很早就有試著去寫子彈筆記了，但是一直用不起來，因為每天都要去記錄，實在是太麻煩了，也沒有動力去做，所以也就逐漸放棄了。這個暑假，我要嘗試去每天寫子彈筆記，雖然不一定會成功，但絕對會有一些收穫。

每週寫一篇文章

暑假時間也比較多，這應該也沒什麼困難，而且不缺題材。

目前應該會寫的文章包含：

  • 最後十四堂星期二的課心得
  • 基度山恩仇記心得
  • 牧羊少年奇幻之旅心得
  • 烏龍派出所系列文章
  • Python 教學系列文章

學習

前端

我有很長一段時間想去學習前端，但最後都半途而廢。所以這次，我要嘗試去學習前端，並學會基礎的東西。btw，我自學的網站是這個，跟我一樣想學前端的人可以參考一下。

目標：

  • 熟悉HTML / CSS / Javascript
  • 學會前端的基本知識

素描

這也是我一直想嘗試的事，但始終沒有去學，雖然平常偶爾會畫一點東西。目標是能畫大部分自己喜歡的東西。

Racket

Racket是一門Lisp方言，學會了雖然沒什麼用，但會有不少啟發。目標是熟悉Racket的語法。

Racket官網

英文

英文，你揮之不去的惡夢。有時候會覺得真的很靠杯，為什麼那些資料都是英文的，為什麼我天殺的必須要去學另ㄧ門語言。但世界就是那麼殘酷，你不會英文，你就等於是只能讀二手的資料。有時候真的很希望我天生就會英文，這樣就方便多了。

目標:

  • 基本的讀、寫、聽
  • 足已讀英文的技術文章
